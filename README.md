# Work Day Scheduler Starter Code

## Purpose
A website that contains a simple daily schedule app that automatically displays the current date and hour and can save and display tasks locally.

## Built With
* HTML
* CSS
    - Bootstrap
* JavaScript
    - jQuery
    - Moment.js
